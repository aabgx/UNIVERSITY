1) Ce este adevarat despre urmatorul cod javascript

	return new Promise((resolve) => {
		setTimeout(resolve, timeout);
	});
}

	await wait(3000);
	console.log('Finished Task');
}
async function executeTasks() {
	await Promise.all([WaitFor1Second(), WaitFor1Second()]);
}
executeTasks();

b) In consola vor aparea 2 "Finished Task" decalate cu aproximativ 3 secunde, deoarece javascript e single thread
c) Va da eroare pentru ca nu putem pornii 2 functii async in acelasi timp

Explicatie: Javascript e single thread, dar avem doua apeluri asincrone de functii.
